Introduction

Python is a high-level, interpreted programming language with a simple syntax. It is used in various applications like web development, data science, automation, and more.

Its simplicity and focus on readability make it ideal for both beginners and advanced programmers.

  • Simple and Readable Syntax: Python stands out for its clean syntax, making it easy to read and write code. Code blocks are defined using indentation instead of braces or keywords.
  • Dynamic Typing: There is no need to explicitly declare variable types; Python infers the data type automatically.
  • Interactivity: Python can be executed interactively in the console, allowing quick testing of code snippets.
  • Widely Used: Popular in areas such as web development, automation, data science, artificial intelligence, and software development in general.
  • Extensive Libraries: Python has a vast standard library and thousands of third-party libraries (such as numpy, pandas, matplotlib, etc.) that extend its capabilities.

Operators

Operators in Python are used to perform operations on variables and values. Arithmetic operators perform mathematical calculations, comparison operators compare values, and logical operators evaluate conditions.

Python supports arithmetic operators (+, -, *, /, %), comparison operators (==, !=, <, >) and logical operators (and, or, not).

Control structures

Python uses if, else, and elif statements to control the flow of the program based on certain conditions. Indentation is crucial for defining code blocks in these structures.

                    
                        if age >= 18:
                        print("Is an adult")
                        else:
                        print("Is not an adult")
Loops

for loop: Used to iterate over sequences (lists, tuples, dictionaries).

                    
                        for i in range(5):
                            print(i)

while loop: Repeats a block of code while the condition is true.

                    
                        counter = 0
                        while counter < 5:
                            print(counter)
                            counter += 1
